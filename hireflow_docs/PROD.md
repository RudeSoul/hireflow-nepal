# HireFlow Nepal - Product Requirements Document

## Vision

Build the modern hiring operating system for Nepali companies.

HireFlow helps organizations manage jobs, candidates, interviews, and hiring workflows in one place.

---

## MVP Goal

Allow companies to:

- Create jobs
- Publish jobs
- Receive applications
- Manage candidates
- Move candidates through hiring stages
- Collaborate with recruiters and hiring managers

---

## User Types

### Company Owner

Can manage company settings and users.

### Recruiter

Can create jobs and manage candidates.

### Hiring Manager

Can review candidates and leave feedback.

---

## MVP Features

### Authentication

- Login
- Signup
- Password reset

### Company Workspace

- Create company
- Manage company profile

### Jobs

- Create job
- Edit job
- Publish job
- Archive job

### Career Page

- Public jobs listing
- Job detail page
- Apply form

### Candidates

- Candidate profile
- Resume upload
- Application history

### Hiring Pipeline

- Applied
- Screening
- Interview
- Offer
- Hired
- Rejected

### Notes

- Internal comments
- Hiring feedback

---

## Out Of Scope

- AI matching
- Interview scheduling
- Analytics
- Viber integration
- Facebook integration

These will be built after MVP validation.
